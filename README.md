## II. laravel10
1. Cd `laravel10`.
2. Run `composer install`.
3. Run `php artisan reset:clear`.
4. Run `php artisan serve`.
   - work : [http://127.0.0.1:8000](http://127.0.0.1:8000).

## III. vue3
1. Cd `vue3`.
2. Cd `npm install`.
3. Cd `npm run dev`.
